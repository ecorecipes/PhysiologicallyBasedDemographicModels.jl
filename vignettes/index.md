# Overview

This book collects tutorial vignettes for
**PhysiologicallyBasedDemographicModels.jl**, a Julia package for
building and analyzing Physiologically Based Demographic Models (PBDMs).

PBDMs are mechanistic, process-based simulation models that link
individual physiology to population dynamics through temperature-driven
development, resource acquisition (supply/demand), and distributed
maturation delays.
